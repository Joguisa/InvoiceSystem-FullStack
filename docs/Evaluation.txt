Implementar una solución para el “Ingreso y gestión de facturas” según lo detallado a continuación:

 Para este diseño se requiere:

Se debe permitir la creación de facturas y maestros de productos, clientes y usuarios (vendedores).
Estos usuarios pueden ingresar facturas que contenga: Identificación del cliente, Nombre Cliente, Teléfono, correo, vendedor, fecha, múltiples formas de pago y detalles de la Factura (Producto, Cantidad, Precio Unitario, Precio total).
El sistema debe permitir la consulta y visualización de las facturas ingresadas.
El sistema debe ofrecer funcionalidades para filtrar y buscar facturas por criterios como número de factura, fecha y monto.
Tomar como ejemplo el siguiente demo:

URL: https://obedalvarado.com/demos/simple_invoice/

Frontend: Frontend Angular v16 o superior (Web SPA)
Backend: C#.Net 6 o superior (API Rest)

Git: La solución debe ser publicada en un repositorio Git y confirmar los cambios de manera constante para validar el avance progresivo, no se tomara en cuenta repositorios que contengan un único commit.
